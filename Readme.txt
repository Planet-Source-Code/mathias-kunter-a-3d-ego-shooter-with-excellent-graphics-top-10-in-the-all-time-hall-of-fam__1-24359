Installation of SHOT IT

Simply extract the contens of this zip file to any directory.
Open the project with VB and click the start button.

You shold have a 400 MHZ CPU and a 3d graphics card for harware rendering.
This game also needs DirectX 7 or higher to run.
Hint: Compile the game before starting it in VB to increase frame rates.

Have fun!
